# desktop-avalonia

Avalonia 桌面 .NET 能力 plugin — 收斂 Avalonia + CommunityToolkit.Mvvm 專屬做法。通用 .NET / git 能力不重複收（見 backend-dotnet、dev-flow）。

## 定位

`[desktop / dotnet]` 任何 Avalonia + CommunityToolkit.Mvvm 桌面專案適用。

## Skills（3）

| Skill | 用途 |
|-------|------|
| `avalonia-mvvm` | Avalonia + CommunityToolkit.Mvvm 慣例（ObservableProperty/RelayCommand、compiled bindings、DI DataContext、設計時支援） |
| `cross-platform` | 跨平台路徑（Path.Combine、OperatingSystem.Is*）與腳本（LF、UTF-8、Windows/Unix 對照） |
| `desktop-build-publish` | 桌面應用建置/發佈（RID、self-contained、single-file、macOS .app/.dmg、桌面捷徑） |
