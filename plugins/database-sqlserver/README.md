# database-sqlserver

SQL Server 資料庫能力 plugin — table/view 撰寫慣例、TableDescription 文件生成、EF Core migration 工具與疑難排解。

## 定位

`[database]` SQL Server + EF Core 專案適用。範例識別符已中性化，多租戶/客戶巡檢等專案專屬邏輯已剔除。

## Skills（6）

| Skill | 用途 |
|-------|------|
| `generating-table-description` | 由 SQL view 產生 TableDescription 文件 |
| `table-conventions` | Table 撰寫慣例 |
| `view-conventions` | View 撰寫慣例 |
| `syncing-db-entities` | EF Core Power Tools re-scaffold Entity、雙 config 同步 |
| `migration-troubleshoot` | Migration 診斷、ModelSnapshot 衝突、回滾 |
| `ef-migration` | EF Core migration 操作 |
