# frontend-vue

Vue 3 + TypeScript + Quasar 前端能力 plugin — 收斂通用的前端架構、composable、除錯、i18n 與 OpenSpec 工作流。

## 定位

`[frontend]` 任何 Vue 3 + TS + Quasar 專案適用。MoldPlan 專屬的 UI 業務約定（base-page、component-library、inline-edit 等）刻意不收，留在各專案層。

| Skill | 用途 |

| `frontend-architecture` | Vue3+TS+Quasar 架構/分層決策 |
| `vue-composables` | 業務邏輯抽 composable 模式 |
| `i18n-implementation` | i18n 違規偵測與 key 組織 |
